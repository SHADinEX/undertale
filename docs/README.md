# Undertale Python Clone

Простая RPG на движке Pygame с коллизиями, диалогами и системой сохранений.

## Установка
1. Установите Python 3.11+
2. Установите зависимости: `pip install -r requirements.txt`

## Управление
- **Стрелочки / WASD** — Движение сердечка.
- **E** — Взаимодействие с NPC / Пролистывание текста.
- **F3** — Включить/выключить сетку отладки (Debug хитбоксов).
- **F5** — Быстрое сохранение игры.
- **F9** — Быстрая загрузка игры.
